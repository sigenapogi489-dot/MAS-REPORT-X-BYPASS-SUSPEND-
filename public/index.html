How to Get Your Facebook State

1

Download Kiwi Browser

Kiwi Browser is required as it's the only mobile browser with extension support.

Download Kiwi Browser
2

Download the C3C FBstate Extension

This extension will help you extract your Facebook state.

Download C3C Extension
3

Install the Extension in Kiwi Browser

  1. Open Kiwi Browser
  2. Tap on the three dots () in the top right corner
  3. Select "Extensions"
  4. Enable "Developer mode" if prompted
  5. Tap on the "+" button or "From file" option
  6. Select the downloaded ZIP file (c3c-fbstate-1.5.zip)
4

Access Your Facebook State

  1. Open Facebook in Kiwi Browser (preferably use a test account)
  2. Log in to your Facebook account
  3. Tap on the three dots () again
  4. Scroll down and tap on "C3C FBSTATE Utility"
  5. Click "Copy to Clipboard"
5

Use Your Facebook State

  1. Return to this page
  2. Paste the copied FBstate value into the "Facebook State" field below
  3. Enter the URL you want to share
  4. Set the desired share count and time interval
  5. Click "Start Sharing" to begin the process
